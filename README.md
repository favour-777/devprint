DevPrint converts GitHub profiles into recruiter-ready developer snapshots. It evaluates repositories, activity, languages, and public metadata to deliver a clear, decision-focused overview in seconds.

Built as my submission for <i>Remote Hustle's</i> Developer Challenge (RHDC).



## Why I Built This

I was recruited for RHDC through GitHub, and that moment stuck with me.

DevPrint started with a simple idea: **what if a GitHub profile alone could answer “should I consider this developer?” instantly?**

Instead of manually scanning repositories, bios, and activity, DevPrint pulls everything together into a single, structured snapshot. It highlights strengths, impact, and contact points so recruiters can move faster and with confidence.



## Who This Is For

- **Recruiters** who need fast, decision-ready insights  
- **Hiring managers** who want a quick evaluation before a deeper review  
- **Developers** who want a clean, shareable profile summary  



## Core Features

- AI-generated recruiter insight  
- Structured profile snapshot with key metadata and contact links  
- Impact score based on stars, forks, followers, and activity  
- Language distribution with visual breakdown  
- Developer archetype classification  
- Reliable summary generation with automatic fallback  
- Shareable report links  



## How It Works

1. Fetches public GitHub profile and repository data  
2. Analyzes language usage and repository activity  
3. Computes an impact score from engagement metrics  
4. Generates a recruiter-focused summary using a serverless AI endpoint (with fallback)
